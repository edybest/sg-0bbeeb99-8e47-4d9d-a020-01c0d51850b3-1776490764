# AMBC CLUB

## Vision
A comprehensive management platform for the AMBC bowling club, streamlining member administration, lane assignments, scoring, tournaments, and real-time WhatsApp integration for member engagement.

## Design
* `--primary`: 222 47% 11% (dark navy)
* `--background`: 0 0% 100% (white)
* `--foreground`: 222 47% 11% (dark navy)
* `--accent`: 38 92% 50% (golden orange)
* `--muted`: 210 40% 96% (light slate)
* `fonts`: Geist (sans), GeistMono (mono)
* `style`: Modern sports club, data-dense but readable, highly mobile-responsive.

## Features
* WhatsApp webhook integration for auto-joining events
* Live game comments and TikTok live overlays
* comprehensive admin dashboard for lane, score, and member management
* PWA support for mobile access
